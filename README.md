# WE4B-Project

## Setup
1. Unzip the project
2. Navigate to the project directory:
```bash
cd WE4B-Project
```
3. Install submodules:
```bash
git submodule update --init --recursive
```
4. Install node dependencies:
```bash
./install-all.sh
```
5. Start both the frontend and backend:
```bash
./start-all.sh
```
6. Open your browser and navigate to:
```url
http://localhost:4200
```
